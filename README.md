# Salesforce Loan Application Risk & Follow-up Management System

## Overview
A Salesforce-based application designed to manage loan applications, assess risk, assign agents, and track follow-ups, inspired by NBFC workflows.

## Features
- Customer profile management
- Loan application lifecycle tracking
- Risk classification
- Agent assignment
- Follow-up and reminder tracking
- Dashboards for approvals and pipeline visibility

## Tech Stack
- Salesforce Lightning Experience
- Custom Objects & Validation Rules
- Salesforce CLI (sf)
- Apex (planned)
- GitHub for version control
